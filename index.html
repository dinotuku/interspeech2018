    Multi-Modal Data Augmentation
    for End-to-end ASR

    Adithya Renduchintala, Shuoyang Ding, Matthew Wiesner,
    Shinji Watanabe

    Johns Hopkins University

    Goal

    Leverage much larger text corpus

    alongside limited amounts of speech datasets

    to improve the performance of end-to-end ASR

    Novelty

    • Traditional decoding process is unchanged
    • Augment the input with text-based synthesized input

    Example sequences of the phrase JOHN BLARE AND COMPANY

    Architecture

    Attention-Based RNN Encoder-Decoder network

    x: acoustic features (83dim)

    80 MFCC + 3 pitch features

    z: synthetic input (83dim)

    Embedding layer is omitted

    Alternate between batches from x and z

    Special Method

    • Pyramidal encoder: down-sample the frame encodings
    • Location-aware attention*
    • RNNLM trained on the same sentences used for augmentation

    *: The previous timestep attention weights $ \alpha_{j-1} $ are "smoothed" by a convolution operation and fed into the attention weight computation. Once attention weights are computed, a weighted sum over encoder hidden states generates the context vector $ c_j $

    Generate Synthetic Input

    Synthetic Input Method
    Charstream Omit word boundaries
    Phonestream CMUDICT --> Train G2P (Phonetisaurus)
    Rep-Phonestream Assume the duration as normal distribution,
    estimate it from frame-level phoneme in TIMIT,
    and then sample a duration from it
    (divided by 4 for down-sampling)
